# Camera Shop
Интернет-магазин фото- и видеотехники.

## Используемые технологии
- TypeScript
- React 
- Vite 
- Redux Toolkit
- Axios
- React Router

## Функционал проекта

### Каталог
- Страница «Каталог» — главная страница сайта. Каталог включает все доступные камеры, каждая из которых представлена на отдельной странице с подробным описанием. 
- Фильтры: пользователи могут фильтровать товары по типу, категории, уровню и цене. Это поможет им сузить поиск до наиболее релевантных товаров.
- Сортировка: товар в каталоге можно сортировать по популярности (по оценкам пользователей) и цене. При помощи стрелок можно менять направление: по возрастанию (от меньшего параметра к большему), по убыванию (от большего к меньшему).
- Пагинация (по 9 товаров на страницу), при переходе по страницам URL обновляется.
- Параметры сортировки и фильтрации имеют свой уникальный URL.
- Живой поиск с автоподсказками. Минимальное количество символов для начала поиска — 3. Выдача товаров (в выпадающем списке) перерисовывается после каждого введённого символа. Удаление введённых данных (х) можно сделать после ввода хотя бы одного символа. В выпадающем списке при клике мышью или с клавиатуры можно выбрать один из предложенных вариантов. При выборе варианта происходит перенаправление на страницу соответствующей карточки товара.
- В шапке на странице «Каталог» размещается слайдер с тремя новинками. Смена слайдов происходит автоматически с интервалом 3 секунды.
- Добавление товара из каталога: из каталога товар добавляется в корзину по клику на кнопку «Купить» в карточке товара и подтверждении действия в попап. По клику на кнопку «Купить» открывается попап «Добавить товар в Корзину». По клику на кнопку в попап «Добавить в корзину» открывается попап «Товар успешно добавлен». По клику «Продолжить покупки» попап закрывается и пользователь остается в каталоге. Кнопка «Купить» меняет свое состояние в карточке товара. По клику на кнопку «В корзине» происходит переход на страницу Корзины.

### Страница камеры `/camera/:id`
- Название камеры в заголовке и хлебных крошках   
- Переключаемые вкладки: Характеристики и Описание  
- Просмотр последних отзывов и кнопка «Показать больше»  
- Слайдер похожих товаров  

### Корзина `/cart`
- Добавление товаров из каталога и со страницы товара  
- Управление количеством товаров (макс. 9)  
- Удаление товаров с подтверждением  
- Сохранение состояния корзины через `localStorage`  
- Применение промокодов и динамические скидки  
- Кнопка «Оформить заказ» с прелоадером и отправкой на сервер  

### Отзывы
- Модальное окно с формой для нового отзыва  
- Валидируемые поля: имя, комментарий, рейтинг  
- После отправки отображается попап «Спасибо за отзыв» и новый отзыв на странице  

### Модальные окна
- Закрытие через Escape, кнопку «Х» или клик на overlay  
- Фокус зациклен на интерактивных элементах  
- Скролл страницы блокируется, фон затемняется  

### Слайдеры и рекламные блоки
- Слайдер акций на главной странице каталога  
- Слайдер похожих товаров на странице товара  
- Навигация через стрелки и буллеты, автоматическая смена слайдов  

### Страница 404
- Корректная маршрутизация на несуществующие страницы  
- Простая реализация: текст 404 и ссылка на главную 

## Запуск проекта

1. Установите зависимости, выполнив команду `npm install`.

2. Запустите приложение, выполнив команду `npm start`.
